Description

The code is follow the instruction of the assignment.

This program can handle delay and packet dropping.

The maximum of delay is 500ms by default, you can change it in UDPClient variable timeOut.

When client try to sending data to server:

  1. Three-way handshake
    1. Client send SYN packet to server
    2. Server send SYN-ACK packet to client
    3. Client send
  2. Client start sending data
    1. Window size is 4
    2. Server will send ACK packet after it receive a packet
  3. Server received all data
  4. Client send a packet which tell server I am ready for receiving response
  5. Server read the request data
  6. Server send response to client
  7. Done

Example of command:

  1. "GET http://localhost:8007/"
  2. "GET http://localhost:8007/1"
  3. "POST --data \"update data\" http://localhost:8007/1"
  4. "POST --data \"param1=value1&param2=value2\" http://localhost:8007/brent2.txt"
  5. "POST --data \"update data\" http://localhost:8007/readonly.txt This is readonly file, so you will received 400 bad request.
